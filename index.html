작동 원리

세 번의 검증을 거친 결론

바로 답하지 않습니다. 세 번 검증한 뒤에야 결론을 꺼냅니다.

01

의도 확정

당신이 진짜 원하는 것을 확인합니다. 성공 조건을 세우고, 최악의 시나리오를 먼저 그립니다. 질문 속 편향도 이 단계에서 감지합니다.

이 결정이 성공하려면?
02

선택지 발산

다양한 아이디어를 대량 생성하고, 역발상과 범위 확장을 포함해 당신이 안 본 경로를 찾습니다.

생각 못한 선택지가 있을까?
03

반론 검증

결론을 먼저 공격합니다. 독립된 검증자가 반례와 논리 결함을 찾고, 살아남은 것만 제시합니다.

이 결론이 틀리려면?
두 가지 모드

같은 구조, 다른 온도

혼자 안고 있던 고민에는 따뜻하게, 팀 앞에 내놓을 결정에는 날카롭게.

상담

개인 고민

이직, 관계, 진로, 인생의 갈림길.

좋은 질문 "3년차 개발자인데 연봉 20% 더 주는 스타트업 오퍼가 왔어. 현재 팀은 좋지만 성장이 정체된 느낌이고, 솔직히 가고 싶은 마음이 70%인데 후회할까봐 무서워."
이렇게 물어보면 달라집니다
이렇게 묻지 마세요 "이직해야 할까?"
이렇게 물어보세요 "3년차인데 연봉 20% 더 주는 스타트업 오퍼가 왔어. 팀은 좋지만 성장 정체, 가고 싶은 마음 70%인데 후회할까봐..."

상황, 선택지, 걸려있는 것, 솔직한 마음을 함께 알려주세요.

컨설팅

조직 의사결정

전략, 채용, 아키텍처, 파트너십.

좋은 질문 "8명 디자인팀에 Figma AI를 전사 도입할지 고민 중이야. 라이선스 비용은 연 2400만원 추가고, 디자이너 절반은 반발 중. 6개월 안에 생산성 20% 향상을 보여줘야 해."
이렇게 물어보면 달라집니다
이렇게 묻지 마세요 "팀에 AI 도구를 도입할까?"
이렇게 물어보세요 "8명 디자인팀에 Figma AI 도입, 비용 연 2400만원, 디자이너 절반 반발, 6개월 내 성과 필요..."

팀 규모, 비용, 기한, 이해관계자 반응을 포함하세요.

특징

한번 내린 결론을 다시 뒤집지 않게

최악의 시나리오 선제 검증

결론이 실패하는 경로를 먼저 그리고, 그 시나리오를 방지하는지 검증합니다.

공유 가능한 결론 리포트

결론을 HTML 리포트로 생성합니다. 저장하고, 공유하고, 인쇄할 수 있습니다.

한 번 쓰고 끝이 아닙니다

이전 결정의 결과를 추적하고, 다음 판단의 정확도에 반영합니다.

정직한 한계 인식

AI의 한계를 숨기지 않습니다. 의료, 법률, 재정 관련 고민에는 전문가 상담을 안내합니다.

자주 묻는 질문

먼저 답해드립니다

내 대화 내용이 어딘가에 저장되나요?
대화 내용은 Claude Code의 보안 정책을 따르며, 갈피는 별도 서버로 데이터를 전송하지 않습니다. 설치 스크립트는 오픈소스이므로 코드를 직접 확인할 수 있습니다.
비용이 드나요?
갈피 자체는 무료입니다. Claude Code 사용량에 따른 API 비용만 발생합니다.
어떤 환경에서 쓸 수 있나요?
Windows + Git Bash + Claude Code가 설치된 환경에서 동작합니다.
AI 결론을 그대로 믿어도 되나요?
갈피는 결론을 내리기 전에 독립된 검증자가 반례와 논리 결함을 먼저 공격합니다. 그래도 틀릴 수 있으므로, 최종 결정은 반드시 본인의 판단으로 내려주세요.
